## Setup

### Dependencies MacOS

```
brew install glfw
brew install lua
brew install luarocks
luarocks install dkjson
brew install pkg-config
```
optional (to build editor)
```
brew install glm
```
## Features

#### Entity Component System
Powered by FLECS

#### Physics Engine
Powered by Box2D

#### Render & Shading
Self-made OpenGL 3 Renderer
